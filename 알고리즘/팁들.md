### 배열
- sort함수는 원본을 변경한다. 

- 배열에서 shift 함수 사용시 효율성 떨어짐
- indexOf가 findIndex보다 빠르다. 

##### 배열.sort
- 원본을 변경한다.

``` javascript
['z', 'a', 'b'].sort() // ['a', 'b', 'z']
```
##### 배열.slice
``` javascript
[1, 2].slice(2) // []
[1, 2].slice(1, 1) // []
```
##### 배열.slice와 배열.splice
``` javascript 
[1,2,3].slice(0, 2) // 0부터 2 전까지 뽑음
[1,2,3].splice(0, 2) // 0부터 2개 뽑음 
```

### 숫자

##### NaN도 숫자 타입이다. 
``` javascript 
typeof Number('a123') // 'number'
Number('a123') // NaN
```

##### NaN비교시 isNaN 사용
인자를 숫자로 형변환 후 판단
``` javascript
isNaN('a') // true;

// Caution! 
isNaN('1e22') // false 
```
