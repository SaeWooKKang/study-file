# range와 take의 재해석

``` javascript
_.go(
  _.range(10), // <-- 0부터 9까지의 배열
  _.take(3), // <-- 앞에서부터 3개만 자르기
  // _.each(log)
);

// 보는 시각을 좀 바꿈
// 어떤 일들이 특정 시간에 걸쳐서 ... 일들을 이터러블로, 리스트로 바라보고 프로그래밍
// 단순히 유틸성으로 다루는 것이 아닌
// 로직을 구성하고 만드는 문법을 대체하는 언어로서 사용할 수 있다
_.go(
  _.range(10), // <-- 0 부터 9까지의 이터러블, 최대 10번 일어나는 일
  L.map(_.delay(100)),
  L.filter(a => a % 2),
  L.map(_ => new Date()),
  L.take(3), // <-- 최대 3개의 값을 필요하고, 최대 3번의 일을 수행 
  _.each(log)
);
```
