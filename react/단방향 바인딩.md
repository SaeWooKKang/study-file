##### 데이터 바인딩 
- 두 데이터를 동기화하는 기법 

#### 양방향, 단방향
> HTML에서 데이터를 변경 시킬수 있냐 없냐로 구분할 수 있다. 
- 양방향
  - HTML -> JS 변경 ok
  - HTML <- JS 변경 ok
- 단방향
  - HTML -> JS 변경 x  
  - HTML <- JS 변경 ok 

#### React는 단방향 데이터 바인딩
- HTML과 js 모두에서 데이터 변경이 가능한 것이 아닌 js에서만 데이터 변경이 가능
- HTML에서 js의 데이터 변경 불가능 
- 이벤트로 역방향 데이터 흐름 추가로 변경 가능
